# E-Commerce Store

A modern e-commerce website built with Next.js, TypeScript, and Context API using FakeStore API.

## Features

- **Framework**: Next.js 16 with TypeScript
- **State Management**: React Context
- **API**: FakeStore API
- **Features**:
  - Product listing
  - Shopping cart
  - Product search/filter
  - Responsive design

## Tech Stack

- Next.js 16
- TypeScript
- Context API
- CSS Modules/Tailwind
- FakeStore API

## Installation

```bash
git clone [repo-url]

cd [next-router]

npm install

npm run dev
